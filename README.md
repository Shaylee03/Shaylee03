# Shaylee

I work on product prototypes that combine desktop interaction, lightweight automation, and human-centered AI experiences.

## Featured Project

### Low-Interruption Desktop AI Robot

A personal product prototype for study, remote work, and knowledge-work scenarios. The project focuses on a practical question:

> When should a desktop assistant step in, stay quiet, delay a reminder, or switch from voice to a lower-interruption channel?

The prototype combines local desktop context sensing, attention-state rules, four-level feedback, memory design, and a downloadable Windows starter kit.

- Portfolio: https://shaylee03.github.io/zsy-desktop-companion/
- Repository: https://github.com/Shaylee03/zsy-desktop-companion
